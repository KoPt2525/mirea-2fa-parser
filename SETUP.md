# Инструкция по настройке с нуля

Здесь описано всё что нужно сделать перед запуском скрипта.  
Три независимых блока — выполняй по порядку.

---

## Блок 1. Google Таблица и API

### 1.1 Создать Google Таблицу

1. Открой [sheets.google.com](https://sheets.google.com) → **Создать**
2. Назови таблицу как угодно, например `MIREA 2FA`
3. В первой строке заполни заголовки:

| A | B | C | D | E |
|---|---|---|---|---|
| ФИО | Email | *(пусто)* | Код | Время когда код пришел |

4. Начиная со строки 2 — заполни данные участников:

| A | B |
|---|---|
| Иванов Иван Иванович | ivanov.i.i@edu.mirea.ru |
| Петров Пётр Петрович | petrov.p.p@edu.mirea.ru |

> **Важно:** в столбце B нужен адрес `@edu.mirea.ru` — именно на него МИРЭА отправляет 2FA-код. Не mail.ru, не gmail.

5. Скопируй **ID таблицы** из адресной строки браузера:
```
https://docs.google.com/spreadsheets/d/  ВОТ_ЭТО_И_ЕСТЬ_ID  /edit
```

---

### 1.2 Создать Google Cloud проект и включить API

1. Открой [console.cloud.google.com](https://console.cloud.google.com)
2. Вверху нажми на название проекта → **Новый проект**
   - Название: `mirea-parser` (любое)
   - Нажми **Создать**
3. Убедись что выбран только что созданный проект (вверху)
4. В левом меню: **APIs & Services → Библиотека**
5. В поиске введи `Google Sheets API` → выбери его → нажми **Включить**

---

### 1.3 Создать сервис-аккаунт и скачать ключ

1. В левом меню: **APIs & Services → Credentials (Учётные данные)**
2. Нажми **+ Create Credentials → Service account**
3. Заполни:
   - **Service account name:** `mirea-parser`
   - Нажми **Create and continue**
   - На следующих шагах ничего не трогай — просто **Done**
4. В списке сервис-аккаунтов нажми на только что созданный
5. Перейди на вкладку **Keys (Ключи)**
6. **Add Key → Create new key → JSON → Create**
7. Файл `.json` скачается автоматически — **не теряй его**, он нужен для скрипта

---

### 1.4 Дать сервис-аккаунту доступ к таблице

1. Открой скачанный JSON-файл любым текстовым редактором
2. Найди строку `"client_email"` — скопируй адрес вида `mirea-parser@...iam.gserviceaccount.com`
3. Открой Google Таблицу
4. Нажми **Настройки доступа** (кнопка «Поделиться» / Share)
5. Вставь скопированный email → выбери роль **Редактор** → **Отправить**

> Теперь скрипт сможет читать и писать в таблицу через этот аккаунт.

## Блок 2. Центральный почтовый ящик

Это ящик `@mail.ru` на который все участники будут пересылать коды.  
Можно использовать существующий ящик или создать новый.

### 2.1 Создать ящик (если нужен новый)

1. Зайди на [mail.ru](https://mail.ru) → **Регистрация**
2. Придумай адрес, например `mirea.codes.2fa@mail.ru`
3. Запомни пароль — он понадобится на следующем шаге

---

### 2.2 Создать App Password (пароль для скрипта)

Mail.ru не позволяет входить по IMAP с основным паролем — нужен отдельный пароль приложения.

1. Войди в созданный ящик на [mail.ru](https://mail.ru)
2. Нажми на аватарку → **Настройки** (или сразу [перейди сюда](https://account.mail.ru/user/security))
3. Раздел **Безопасность**
4. Найди **Пароли для внешних приложений** → нажми **Добавить**
5. Введи название: `IMAP parser` → нажми **Продолжить**
6. Появится пароль вида `xXxXxXxXxXxXxXxx` — **скопируй его сразу**, он показывается один раз
7. Этот пароль — `central_password` в конфиге скрипта

---

### 2.3 Проверить что IMAP включён

1. В настройках ящика → **Почтовые программы**
2. Убедись что переключатель **"Доступ по протоколу IMAP"** включён
3. Сохрани если изменял

---

## Блок 3. Пересылка у участников

Каждый участник должен один раз настроить автоматическую пересылку  
писем от `sso@mirea.ru` на центральный ящик.

### 3.1 Через mail.ru (если личная почта на mail.ru)

> Подходит если участник пользуется `@mail.ru`, `@bk.ru`, `@list.ru`, `@inbox.ru`  
> и на неё уже приходят письма от МИРЭА.

1. Войти в свой ящик на [mail.ru](https://mail.ru)
2. Настройки (шестерёнка) → **Правила фильтрации**
3. Нажать **Создать правило**
4. Заполнить:
   - **От кого:** `sso@mirea.ru`
   - **Действие:** ✅ **Переслать на адрес** → вписать адрес центрального ящика
   - Галку «Оставить копию в ящике» — по желанию
5. Нажать **Сохранить**

После этого все письма от `sso@mirea.ru` будут автоматически улетать на центральный ящик.

---

### 3.2 Через студенческий портал МИРЭА (если адрес @edu.mirea.ru)

Если у участника нет отдельного mail.ru и коды приходят напрямую на `@edu.mirea.ru`:

1. Войти на [lk.mirea.ru](https://lk.mirea.ru)
2. В профиле найти раздел **Почта** или **Email**
3. Проверить на какой адрес привязана учётная запись
4. Если это `@edu.mirea.ru` → зайти в [почту edu.mirea.ru](https://mail.mirea.ru) 
   (работает как обычный mail.ru)
5. Настроить правило фильтрации по инструкции выше (п. 3.1)
